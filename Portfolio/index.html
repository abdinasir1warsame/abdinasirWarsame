Project 1: Gazzeteer

Project Overview :
Welcome to Gazzeteer, your premier global exploration platform! Gazzeteer is a web application designed to offer users an immersive journey into countries worldwide. By integrating various APIs covering geolocation, weather updates, news feeds, and more, Gazzeteer delivers a comprehensive experience for users interested in discovering diverse cultures and landscapes across the globe. Developed using HTML, CSS, JavaScript, and Leaflet.js for map integration, Gazzeteer focuses on creating an engaging environment where users can interactively explore and learn about different countries.
Functionality:
Gazzeteer provides several key features, including an interactive map display powered by Leaflet.js, showcasing countries, cities, and airports visually. It presents essential country information such as population, area, capital, language, and timezone, along with national flags and basic geographical data. Additionally, Gazzeteer offers real-time weather updates and a 7-day forecast, fetches top news articles relevant to selected countries, facilitates currency conversion with live exchange rates, and integrates Wikipedia summaries for deeper insights into countries' history, geography, and culture.
How to Use:
Using Gazzeteer is simple and intuitive. Users can explore the map to discover countries, cities, and airports, accessing detailed information with a click. They can select countries from the dropdown menu to view comprehensive details, check weather conditions and forecasts, read relevant news articles, perform currency conversions, and explore Wikipedia summaries for enriched knowledge.
