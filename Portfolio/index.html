Project 1: Gazetteer

Project Overview :
Welcome to gazetteer, your premier global exploration platform! gazetteer is a web application designed to offer users an immersive journey into countries worldwide. By integrating various APIs covering geolocation, weather updates, news feeds, and more, gazetteer delivers a comprehensive experience for users interested in discovering diverse cultures and landscapes across the globe. Developed using HTML, CSS, JavaScript, and Leaflet.js for map integration, gazetteer focuses on creating an engaging environment where users can interactively explore and learn about different countries.
Functionality:
gazetteer provides several key features, including an interactive map display powered by Leaflet.js, showcasing countries, cities, and airports visually. It presents essential country information such as population, area, capital, language, and timezone, along with national flags and basic geographical data. Additionally, gazetteer offers real-time weather updates and a 7-day forecast, fetches top news articles relevant to selected countries, facilitates currency conversion with live exchange rates, and integrates Wikipedia summaries for deeper insights into countries' history, geography, and culture.
How to Use:
Using gazetteer is simple and intuitive. Users can explore the map to discover countries, cities, and airports, accessing detailed information with a click. They can select countries from the dropdown menu to view comprehensive details, check weather conditions and forecasts, read relevant news articles, perform currency conversions, and explore Wikipedia summaries for enriched knowledge.

Project 2: Company Directory

Project Overview:
Welcome to the Organizational Management System! This web application is designed to streamline the management of departments, locations, and employees within an organization. By leveraging modern web technologies and a user-friendly interface, the system empowers administrators and users to efficiently handle various organizational aspects. Developed using HTML, CSS, JavaScript (jQuery), Bootstrap, PHP, and MySQL/MariaDB, the Organizational Management System focuses on enhancing organizational efficiency and productivity.
Key Features:
The system provides seamless management of departments, locations, and employees, offering functionalities such as adding, editing, and deleting records. Error handling mechanisms ensure data integrity and smooth operation, while search functionalities enable users to quickly locate relevant information. Integration with AJAX enables asynchronous communication with the server, facilitating real-time data retrieval and manipulation without page refreshes.
Technical Stack:
- Frontend Technologies: HTML, CSS, JavaScript (jQuery), Bootstrap - Backend Technologies: PHP, MySQL/MariaDB
Functionality Flow:
Users interact with a clean and intuitive interface, presented with options to manage departments, locations, and employees. Modals provide a structured approach for adding, editing, or deleting records, while comprehensive error handling mechanisms ensure smooth operation. AJAX integration facilitates seamless communication with the server, enhancing user experience and productivity.
The Organizational Management System provides a robust platform for efficiently managing departments, locations, and employees within an organization. With its user-friendly interface, seamless integration of frontend and backend technologies, comprehensive error handling mechanisms, and advanced employee management functionalities, the system empowers users to streamline organizational processes and enhance productivity.

Project 4: Weather App
